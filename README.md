### 2. random_module.py
A dice rolling game that simulates rolling two dice repeatedly until "snake eyes" (both dice showing 1) is rolled.

**Features:**
- Simulates two 6-sided dice rolls
- Continues rolling until snake eyes (total = 2) is achieved
- Prints feedback for each unsuccessful attempt

**Usage:**
```bash
python random_module.py
```

### 3. wiki.py
A Wikipedia API integration script that retrieves and displays a summary of a specified topic.

**Features:**
- Uses the `wikipedia` library to fetch article summaries
- Currently configured to display information about Python programming language
- Returns a concise 2-sentence summary

**Requirements:**
```bash
pip install wikipedia
```

**Usage:**
```bash
python wiki.py
```
